Caminho aleat√≥rio¬∂

Defini√ß√£o¬∂

Um caminho aleat√≥rio consiste no percurso a uma ou mais dimens√µes efectuado em passos discretos em que o sentido e ou direc√ß√£o e eventualmente o comprimento de cada passo s√£o obtidos aleatoriamente de um conjunto ou intervalo predefinido.

Na figura em baixo √© poss√≠vel observar um caminho aleat√≥rio a duas dimens√µes numa rede quadrada com 10 000 passos.

O caminho aleat√≥rio simples¬∂

Um caminho aleat√≥rio simples, como o exemplo anterior, √© um caminho aleat√≥rio em que cada passo √© independente do passo anterior. Nestes casos, embora seja imposs√≠vel saber a traject√≥ria e ou posi√ß√£o final de um caminho aleat√≥rio, √© poss√≠vel saber analiticamente a posi√ß√£o final m√©dia, assim como outras caracter√≠sticas, de um n√∫mero elevado de caminhos aleat√≥rios simples.

De facto sabe-se que o vector posi√ß√£o final √© obtido nas seguintes condi√ß√µes:

  • √â a soma dos vectores de cada passo do caminho.
  • A probabilidade de sair um dado vector em cada passo √© independente dos passos anteriores.
  • A adi√ß√£o de vectores √© comutativa, ou seja independente da ordem em que os vectores saem.

Considerando uma rede quadrada a posi√ß√£o final de um caminho aleat√≥rio simples com n passos √© dada por:

$\begin{cases} \vec{r} = i (\hat{x}) + j (\hat{y}) + k (-\hat{x}) + l (-\hat{y}) \\ n = i + j + k + l \end{cases}$

sendo i, j, k e l o n√∫mero de passos na direc√ß√£o +x, +y, -x e -y respectivamente.

Sabendo a probabilidade de se obter um passo em cada direc√ß√£o √© poss√≠vel conhecer a probabilidade de cada conjunto de valores (i, j, k, l). Aten√ß√£o que n√£o √© o mesmo que dizer conhecer a probabilidade de uma posi√ß√£o final pois a mesma posi√ß√£o final pode ser obtida por mais do que um conjunto de valores (i, j, k, l), ou seja √© necess√°rio somar as probabilidades de todos os conjuntos de valores que levem √† mesma posi√ß√£o final.

Distribui√ß√£o multinominal¬∂

A probabilidade de se obter um dado conjunto de valores (i, j, k, l) √© dada pela distribui√ß√£o multinominal:

$p_{i, j, k, l} = \frac{\left(i + j + k + l\right)!}{i! j! k! l!}p_{+x}^i.p_{+y}^j.p_{-x}^k.p_{-y}^l$

n = i + j + k + l

$p_{+x} + p_{+y} + p_{-x} + p_{-y} = 1$

com $p_{+x}$ a probabilidade de o passo ser na direc√ß√£o +x e assim sucessivamente.

Considerando o caminho aleat√≥rio em que todas as direc√ß√µes t√™m a mesma probabilidade ficamos com:

$p_{i, j, k, l} = \frac{\left(i + j + k + l\right)!}{i! j! k! l!}\left(\frac{1}{4}\right)^i.\left(\frac{1}{4}\right)^j.\left(\frac{1}{4}\right)^k.\left(\frac{1}{4}\right)^l = \frac{\left(i + j + k + l\right)!}{i! j! k! l!}\left(\frac{1}{4}\right)^n$

Podemos assim calcular o vector final m√©dio, a dist√¢ncia m√©dia em rela√ß√£o √† origem e vari√¢ncias em xx e yy.

Valores m√©dios¬∂

Os valores m√©dios de uma grandeza podem ser calculados pela express√£o:

$\bar{X} = \sum_{i=1}^nX_i.p_{X_i}$ com $\sum_{i = 1}^np_{x_i} = 1$

No caso do caminho aleat√≥rio simples com 100 passos podemos aplicar a express√£o anterior para calcular o valor m√©dio de x, y e do vector final.

def inicia(nPassos, nAmostras):
    
    '''Faz a inicializa√ß√£o das estruturas de dados que v√£o armazenar os resultados
    das v√°rias amostragens de caminhos aleat√≥rios
    nPassos: O n√∫mero de passos aleat√≥rios do caminho
    nAmostras: O n√∫mero de amostragens a efectuar
    return: Tupel com os diferentes arrays'''
    
    tipo = 'float'
    
    sx = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em x no passo i
    sy = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em y no passo i
    sr = np.zeros((nPassos + 1, 2), dtype = tipo) #A soma dos vectores posi√ß√£o no passo i
    sx2 = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em x ao quadrado no passo i
    sy2 = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em y ao quadrado no passo i
    sr2 = np.zeros(nPassos + 1, dtype = tipo) #A soma dos vectores posi√ß√£o ao quadrado no passo i
    sNorma = np.zeros(nPassos + 1, dtype = tipo) #A soma das dist√¢ncias √† origem no passo i
    rf = np.zeros((nAmostras, 2), dtype = tipo) #A posi√ß√£o final em cada amostagem
    
    return sx, sy, sr, sx2, sy2, sr2, sNorma, rf

def fazCaminho(nPassos, data):
    '''Faz um caminho aleat√≥rio registando os valores em cada passo na estrutura de dados
    nPassos: N√∫mero de passos do caminho
    data: Estrutura de dados para armazenar os valores
    return: O caminho aleat√≥rio'''
    
    caminho = np.zeros((nPassos + 1, 2), dtype = 'int') #Inicializa o caminho
    #Cria o caminho aleat√≥rio e guarda os dados em cada passo
    for i in range(1, nPassos + 1):
        
        passo = passoSimples()
        
        caminho[i] = caminho[i - 1] + passo
        
        data[0][i] += caminho[i, 0] #sx para cada passo 0..100 a soma do valor em x das nAmostragens
        data[1][i] += caminho[i, 1] #sy
        data[2][i] += caminho[i] #sr
        data[3][i] += caminho[i, 0] ** 2 #sx2
        data[4][i] += caminho[i, 1] ** 2 #sy2
        data[5][i] += np.sum(caminho[i] * caminho[i]) #sr2
        data[6][i] += np.sum(caminho[i] * caminho[i]) ** (1/2) #sNorma
    
    return caminho

def amostragemCaminhos(nPassos, nAmostragens, grafico = True):
    '''Cria nAmostragens de caminhos aleat√≥rios com nPassos

def passoPreferencial(pi, pj, pk, pl):
    '''Calcula um passo de um caminho aleat√≥rio em que as direc√ß√µes +x, +y, -x e -y tem os
    pesos dados pelos argumentos da fun√ß√£o.
    pi: Peso da direc√ß√£o +x
    pj: Peso da direc√ß√£o +y
    pk: Peso da direc√ß√£o -x
    pl: Peso da direc√ß√£o -y
    
    return: O vector com a direc√ß√£o do passo'''
    
    total = pi + pj + pk + pl #N√∫mero total dos pesos
    
    possiveis = np.array(pi * [[1, 0]] + pj * [[0, 1]] + pk * [[-1, 0]] + pl * [[0, -1]]) #Constroi o array com todas as hp√≥teses
    
    direccao = np.random.randint(total)
    
    return possiveis[direccao]

def fazCaminho(nPassos, data, tipo = None):
    '''Faz um caminho aleat√≥rio registando os valores em cada passo na estrutura de dados
    nPassos: N√∫mero de passos do caminho

Caminho aleat√≥rio que se auto evita¬∂

Um caminho aleat√≥rio que j√° n√£o cumpre as defini√ß√µes de caminho aleat√≥rio simples e que j√° n√£o permite uma an√°lise anal√≠tica dos resultados esperados √© o caminho aleat√≥rio que se auto evita.

Este caminho termina quando volta a uma posi√ß√£o que j√° tinha estado anteriormente.

Este caminho cria mais uma vari√°vel de interesse que √© quantos passos cada caminho na amostragem deu antes de terminar.

Vamos assim alterar mais uma vez algumas das fun√ß√µes j√° usadas para produzir e analisar estes caminhos aleat√≥rios.

def inicia(nPassos, nAmostras):
    
    '''Faz a inicializa√ß√£o das estruturas de dados que v√£o armazenar os resultados
    das v√°rias amostragens de caminhos aleat√≥rios
    nPassos: O n√∫mero de passos aleat√≥rios do caminho
    nAmostras: O n√∫mero de amostragens a efectuar
    return: Tupel com os diferentes arrays'''
    
    tipo = 'float'
    
    sx = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em x no passo i
    sy = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em y no passo i
    sr = np.zeros((nPassos + 1, 2), dtype = tipo) #A soma dos vectores posi√ß√£o no passo i
    sx2 = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em x ao quadrado no passo i
    sy2 = np.zeros(nPassos + 1, dtype = tipo) #A soma das posi√ß√µes em y ao quadrado no passo i
    sr2 = np.zeros(nPassos + 1, dtype = tipo) #A soma dos vectores posi√ß√£o ao quadrado no passo i
    sNorma = np.zeros(nPassos + 1, dtype = tipo) #A soma das dist√¢ncias √† origem no passo i
    rf = np.zeros((nAmostras, 2), dtype = tipo) #A posi√ß√£o final em cada amostagem
    tamanho = np.zeros(nAmostras, dtype = tipo) #O n√∫mero de passos realizados antes do caminho terminar
    dim = np.zeros(nPassos + 1, dtype = tipo) #O n√∫mero de caminhos que chegou ao passo i, para dividir os arrays ligados
    #aos passos e obter m√©dias
    
    return sx, sy, sr, sx2, sy2, sr2, sNorma, rf, tamanho, dim

def fazCaminhoEvita(nPassos, data):
    '''Faz um caminho aleat√≥rio registando os valores em cada passo na estrutura de dados
    nPassos: N√∫mero de passos do caminho
    data: Estrutura de dados para armazenar os valores
    return: O caminho aleat√≥rio'''
    caminho = np.zeros((nPassos + 1, 2), dtype = 'int') #Inicializa o caminho
    data[9][0] += 1 #A posi√ß√£o inicial √© efectuada todos os caminhos
    #Cria o caminho aleat√≥rio e guarda os dados em cada passo
    for i in range(1, nPassos + 1):
        
        passo = passoSimples()
        posicaoTmp = caminho[i - 1] + passo
        visitado = False
        for j in range(0, i + 1):
            visitado = caminho[-nPassos + j, 0] == posicaoTmp[0] and caminho[-nPassos + j, 1] == posicaoTmp[1]
            if visitado:
                break
        if visitado:
            break
        
        caminho[i] = posicaoTmp
        
        data[0][i] += caminho[i, 0] #sx
        data[1][i] += caminho[i, 1] #sy
        data[2][i] += caminho[i] #sr
        data[3][i] += caminho[i, 0] ** 2 #sx2
        data[4][i] += caminho[i, 1] ** 2 #sy2
        data[5][i] += np.sum(caminho[i] * caminho[i]) #sr2
        data[6][i] += np.sum((caminho[i] * caminho[i])) ** (1/2) #sNorma
        data[9][i] += 1
    
    return caminho, i

def amostragemCaminhos(nPassos, nAmostragens, grafico = True):

def passoPerpendicular(passoAnterior = 0):
    
    '''Produz um passo num caminho aleat√≥rio perpendicular ao passo anterior
    passoAnterior: Direc√ß√£o do passo anterior, 1 se o anterior foi horizontal,
    -1 se foi vertical e 0 se √© o primeiro passo'''
    
    if passoAnterior > 0:
        direccao = np.random.randint(2)
        possiveis = np.array([[0, 1], [0, -1]], dtype = 'int')
        passoAnterior = -1
    elif passoAnterior < 0:
        direccao = np.random.randint(2)
        possiveis = np.array([[1, 0], [-1, 0]], dtype = 'int')
        passoAnterior = 1
    else:
        direccao = np.random.randint(4)
        possiveis = np.array([[1, 0], [0, 1], [-1, 0], [0, -1]], dtype = 'int') 
        passoAnterior = 1 if (direccao % 2) == 0 else -1
    
    return possiveis[direccao], passoAnterior
